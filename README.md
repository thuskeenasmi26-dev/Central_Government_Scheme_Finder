# AI Government Scheme Finder

AI Government Scheme Finder is a simple web application that helps users find central government schemes they may be eligible for. Users answer a few questions, and the application recommends suitable schemes along with eligibility details, required documents, and where to apply.

## Features

- AI-inspired government scheme recommendations
- Simple questionnaire-based eligibility check
- Displays scheme details and benefits
- Shows required documents
- Provides application guidance
- Voice input using Speech Recognition
- Text-to-Speech support for accessibility
- Multi-language support (English, Hindi, Tamil)
- Responsive design
- Works offline after loading in the browser

## Tech Stack

- HTML5

- JavaScript
- Web Speech API (Speech Recognition & Speech Synthesis)

## Usage

1. Click **Start**.
2. Answer the questions about your profile.
3. View the recommended government schemes.
4. Check eligibility, benefits, and required documents.
5. Use voice input or listen to scheme details with Text-to-Speech.
